# Snake in x64 asm

This is an implementation of Snake in x64 assembly! I wrote it to get more
comfortable with x64 assembly, and it was a lot of fun.

It is written for Windows and will not work on another platform.

You can use `build.bat` to build it, and `run.bat` to run it.

Have fun, and see if you can get the "Snake god" rating.
